* Fonts
the page uses, basically, 3 fonts.
- font(s) for normal text (in this case, Roboto or the default sans-serif font)
- twemoji.ttf: twitter emoji font (from https://github.com/mozilla/twemoji-colr)
- twitma.ttf: the special characters from twitter's new Chirp font:
 - U+E000 (bent arrow (non-unicode))
 - U+EA00 (twitter logo (non-unicode))
 - U+F8FF (apple logo (non-unicode))
 - U+1F503 (refresh symbol (unicode))
